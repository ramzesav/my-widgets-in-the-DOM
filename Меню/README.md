# Меню

## Описание

При клике на пункты меню первого уровня (синий фон) они просто должны раскрывать, либо схлопываться, если уже раскрыты. При клике на пункты меню второго вровня (серый фон) в дальнейшем будет реализовано отображение товаров в каталоге, сейчас пока название группы просто выводится в консоль:
